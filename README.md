# Bank Management System (Java)

## Description
This is a console-based Bank Management System built using Core Java and OOP concepts.

## Features
- Create Account
- Login Authentication
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History

## Technologies Used
- Java
- OOP
- Collections Framework
- LocalDateTime API

## How to Run
1. Clone the repository
2. Compile all files:
   javac *.java
3. Run:
   java Main
